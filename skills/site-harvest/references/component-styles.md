# Component Style Catalogue (Exhaustive)

**Capture EVERY visual pattern. This prevents the "footer links unstyled" problem.**

For EACH of these elements, extract computed styles:

## Navigation

- Header container (background, height, shadow, position)
- Logo (size, spacing)
- Nav links (font, colour, hover states, active states)
- Mobile hamburger (icon, position, animation)
- Dropdown menus (background, shadow, animation)

## Footer

- Footer container (background, padding, layout)
- Footer columns (width, gap, alignment)
- Footer headings (font, colour, size)
- Footer links (font, colour, hover colour, underline)
- Footer social icons (size, colour, spacing)
- Copyright text (font, colour, alignment)

## Typography

- H1-H6 (font-family, size, weight, line-height, colour, margin)
- Paragraphs (font, size, line-height, colour)
- Links (colour, hover, underline, visited)
- Lists (bullet style, spacing, indent)
- Blockquotes (border, background, font-style)

## Buttons & CTAs

- Primary button (background, colour, padding, radius, shadow)
- Primary hover (background change, transform, transition)
- Secondary button (border, background, colour)
- Ghost button (transparent, border only)
- CTA sections (full-width, background, padding)

## Sections & Containers

- Section padding (top, bottom, responsive)
- Container max-width
- Section backgrounds (colours, gradients, images)
- Alternating section patterns

## Dividers (CRITICAL - Previously Missed)

- `<hr>` elements (border style, colour, width, margin)
- Border-based dividers (border-top/bottom on sections)
- SVG wave dividers (capture the actual SVG paths!)
- Angle/diagonal dividers (clip-path, transform)
- Spacing-only dividers (margin/padding patterns)
- Background colour transitions between sections

**For SVG dividers:**
- Extract exact SVG markup
- Note fill colours
- Note positioning (top/bottom of section)
- Note if flipped/mirrored variants exist

## Cards & Boxes

- Card container (background, border, radius, shadow)
- Card hover (transform, shadow change)
- Card image (aspect ratio, object-fit)
- Card content (padding, alignment)
- Card title/text (typography)

## Icons

- Icon source (SVG inline, font icons, image)
- Icon sizes (small, medium, large)
- Icon colours (inherit, specific)
- Icon spacing (margin around icons)
- **IMPORTANT: Download exact SVGs, don't substitute!**

## Forms

- Input fields (border, radius, padding, focus state)
- Labels (font, position, colour)
- Placeholder text (colour, font-style)
- Submit buttons (matches button styles?)
- Error states (border colour, message styling)

## Media

- Image containers (aspect ratios, object-fit)
- Image captions (font, alignment)
- Video containers (aspect ratio, controls styling)
- Galleries (grid layout, gap, lightbox)

## Sidebars

- Sidebar width (desktop)
- Sidebar position (left/right)
- Sidebar widgets (padding, borders)
- Sidebar headings (typography)
- Sidebar collapse behaviour (tablet/mobile)

## Downloads & Attachments

- Download links (icon, text styling)
- File type indicators
- Download buttons vs inline links
